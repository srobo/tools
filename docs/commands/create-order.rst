create-order
============

Synopsis
--------

``sr create-order [-h] <dir> -<n> <schematic> -<m> <schematic> ...``

Description
-----------

Create an order.

Options
-------

--help, -h
    Display help and exit.

Examples
--------

.. code::

    $ sr create-order -2 servo-v4-hw/servo-4-output.sch
